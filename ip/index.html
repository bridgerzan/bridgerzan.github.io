 Command Line Interface (CLI)
 Linux/macOS (curl)
$ curl -s "https://bridgerzan.github.io/ip?format=json"
Get complete JSON response with all IP information
$ curl -s "https://bridgerzan.github.io/ip?format=text"
Get only the IP address in plain text
 Windows PowerShell (irm)
PS> irm "https://bridgerzan.github.io/ip?format=json" | ConvertFrom-Json
PowerShell: Get and parse JSON response
PS> (irm "https://bridgerzan.github.io?format=json").ip
Extract only the IP address from response
 Advanced Options
$ curl -s "https://bridgerzan.github.io?webrtc=true"
Include WebRTC detection results
$ curl -s "https://bridgerzan.github.io?geo=true&browser=true"
Get specific data sections only
 Alternative Methods
$ wget -qO- "https://bridgerzan.github.io?format=json"
Using wget command
$ fetch -q "https://bridgerzan.github.io?format=text"
Using fetch command (BSD systems)
